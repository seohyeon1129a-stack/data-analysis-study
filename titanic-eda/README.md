# Titanic EDA

## 1. 문제 정의
타이타닉 데이터에서 생존 여부에 영향을 준 요인을 탐색한다.

## 2. 데이터 구조
- 총 891명
- 12개 변수
- Age, Cabin, Embarked에 결측치 존재

## 3. 주요 분석

### 3-1. 성별과 생존
- 남성 탑승객 수가 더 많음
- 여성 생존자 수가 더 많음
- 여성 생존율이 압도적으로 높음

### 3-2. 객실 등급과 생존
- 1등석일수록 생존율이 높음
- 성별 효과가 객실 등급보다 더 강하게 보임

## 4. 해석
여성과 어린이를 우선 구조했다는 역사적 배경과 연결될 가능성이 있다.

## 5. 느낀 점
count, sum, mean의 차이를 명확히 이해하는 것이 중요하다고 느꼈다.
